SVGs in Web

Scalable Vector Graphic

  • Defines graphics through XML
  • Perfectly Scales regardless of size.
  • Can be animated.
  • Can be created in a text editor or with a drawing app like Inkscape.

Using SVGs in web

There are two primary ways to utilize SVGs in web.

  • Use a .svg file as an image
  • Use an inline svg in your html

Using an SVG File

Use it like you would any image or background image.

Optimizing Images

Importance of Image Optimization

Most users expect web pages to load within 2 seconds. An estimated 40% will leave if it take longer than 3 seconds.

Image performance and optimization is critical.

Optimization Tools

  • Photoshop
  • TinyJPG | TinyPNG
  • Kraken.io
  • ImageOptim (Mac)

PNG Optimization

PNG images are high quality images that are capable of handling transparency.

  • If you don't need transparency, use a JPEG
  • Use for logos, icons, etc.
  • Transparency allows for a lot of optimization.
  • Cant change quality for optimization.

JPEG Optimization

JPEGS are the most common image type on the web and provide a lot of opportunities for optimization.

  • Use for anything that doesn't require transparency. (Product images, hero images, etc)
  • Can change quality for optimization.
        Show in photoshop

SVG Optimization

SVGs are XML files. Tools can help shave down the size of them and remove redundant paths.

  • Illustrator, InkScape, Sketch plugins
  • Gulp and Grunt
